# Go-To-Market & Liquidity

## Principle
Architecture is broad; launch is dense. North star = **bookings completed per active region per month** (proxy: qualified bids per active inventory unit).

## Wedge
Spokane + Coeur d'Alene + Boise corridor. Categories first: food trucks, farmers/holiday markets, county fairs, community festivals, pop-up retail.

## Minimum viable liquidity (per region)
- Supply: 25–50 hosts, 100–250 inventory units, 10–20 active opportunities, dates within 30–90 days.
- Demand: 100–300 bidders, segmented by category.

## Loops
- Host-to-bidder: hosts share listing links to vendor lists → bidders sign up.
- Review loop: completed bookings → reviews → trust → conversion.
- Data loop: bids/bookings → pricing intelligence → better matches.

## Early monetization
Free host onboarding for early partners; transaction fee on booking only; bidders free early. Featured placement + host subscription later. **Platform fee = 10% seller-side commission, locked as D018** (`DEFAULT_PLATFORM_FEE_BPS = 1000`); O1 is resolved.
